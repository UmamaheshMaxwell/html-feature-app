The Background Clip Property

The background-clip property defines how far the background should extend with in element

Background-clip: border-box (this is default one)

The background extends behind the border

Background-clip: padding-box

The background extends to the inside edge of the border

Background-clip: content-box

The background extends to the edge of the content box

The background-clip property is not supported in internet-explorer 8 and earlier versions;
